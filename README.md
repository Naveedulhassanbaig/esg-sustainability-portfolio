# ESG & Sustainability Portfolio

A professional ESG data analysis portfolio built with Python, Pandas, and Plotly.
Designed for Upwork clients in ESG consulting, investment, and sustainability reporting.

## 6 Projects

| # | Project | Framework | Client Type |
|---|---------|-----------|------------|
| 1 | SASB Industry-ESG Mapping | SASB | ESG Consultants |
| 2 | Sustainability Report Analyzer | GRI | Law firms, NGOs |
| 3 | GRI Compliance Dashboard | GRI | Corporates |
| 4 | ESG Score Calculator | MSCI/Sustainalytics | Investment firms |
| 5 | Carbon Footprint Benchmarker | TCFD/GHG Protocol | Climate consultants |
| 6 | UN SDG Contribution Tracker | UN SDGs | Govt, Impact investors |

## Coverage

- 11 SASB Sectors, 77 Industries, 27 ESG Topics
- 20 Global Companies (Apple, Tesla, ExxonMobil, Unilever...)
- 30 GRI Standards
- 17 UN Sustainable Development Goals
- Scope 1, 2, 3 Carbon Emissions (2018-2023)

## Tools

Python, Pandas, Plotly, Jupyter Notebook, Web Scraping

## Run Locally

```bash
pip install pandas plotly nbformat
jupyter notebook esg_upwork_portfolio.ipynb
```

## Frameworks Covered

- SASB (Sustainability Accounting Standards Board)

- UN SDGs (Sustainable Development Goals)
- GHG Protocol (Scope 1, 2, 3)
